# Docs Folder — Read This First

This folder contains the business and legal document drafts for The Language Blueprint LLC:

- `operating-agreement.md` — single-member LLC operating agreement
- `client-consulting-agreement.md` — the contract you'll use with paying clients
- `data-processing-addendum.md` — DPA (FERPA-aligned data handling terms), referenced as Exhibit C of the consulting agreement
- `district-proposal-template.md` — one-page proposal template for district/school engagements
- `invoice-template.md` — invoice template
- `insurance-contract-checklist.md` — insurance + contract readiness checklist
- `compliance-binder-checklist.md` — overall compliance binder tracker
- `linkedin-update.md` — LinkedIn headline + About section copy

## Important — please read before using these

These documents are **drafted templates**, not a substitute for legal advice. Before you sign anything, send a client contract, or rely on the liability/insurance language:

1. **Have a licensed New York attorney review the Operating Agreement, Client Consulting Agreement, and DPA.** This matters most for the limitation-of-liability clause, indemnification language, and anything tied to FERPA — a small error there is exactly the kind of thing insurance and courts care about.
2. **Talk to an insurance broker** about General Liability, Professional Liability (E&O), and Cyber Liability before you sign your first paying client — several of these documents (especially the consulting agreement's liability clause) assume that coverage is in place.
3. **Fill in every bracketed placeholder** (phone number, EIN, banking info, etc.) — search each file for `[` to find them.
4. Treat the Operating Agreement especially carefully — it's the document that protects your personal assets from business liability. Getting it reviewed is inexpensive relative to what it protects.

None of this blocks you from using these as your working drafts today — just don't treat them as final until a professional has looked them over.
